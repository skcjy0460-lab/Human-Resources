# 🏥 병원 인사관리 시스템
**Hospital HR Management System — Streamlit 기반 근무표 분석 보고서**

---

## 📌 주요 기능

| 탭 | 기능 |
|---|---|
| 📋 근무표 | 월별 전체 근무표 (색상 배지, 일별 근무 현황 차트) |
| 📊 통계 분석 | 근무 유형 분포, 부서별 비교, 연차 소진율 게이지 |
| 👤 직원별 현황 | 직원 개인 상세 정보 및 개인 캘린더 |
| ⚠️ 알림/이슈 | 연차 부족, 병가 다수, 야간 과다 자동 감지 |
| 📥 다운로드 | CSV/Excel/보고서 다운로드, 샘플 양식 제공 |

---

## 🚀 실행 방법

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 앱 실행
```bash
streamlit run app.py
```

### 3. 브라우저 접속
```
http://localhost:8501
```

## 📂 엑셀 파일 형식

### 필수 컬럼
| 컬럼명 | 설명 | 예시 |
|---|---|---|
| 사번 | 직원 ID | EMP001 |
| 성명 | 직원 이름 | 김민준 |
| 부서 | 소속 부서 | 간호팀 |
| 직책 | 직급/직책 | 수간호사 |
| 입사일 | 입사 날짜 | 2020-03-01 |
| 연차발생일수 | 해당 연도 연차 일수 | 15 |
| 1일 ~ 31일 | 날짜별 근무 유형 | 근무 / 야간 / 오프 |

### 근무 유형 입력값
```
근무  / 야간  / 오프  / 연차
반차(오전) / 반차(오후) / 병가 / 공휴일
```

## ⚙️ 자동 감지 알림

- **연차 부족 경고**: 소진율 30% 미만 직원
- **병가 다수**: 3일 이상 병가 사용 직원  
- **야간 과다**: 평균의 1.5배 초과 직원
- **최소 인원 미달**: 평일 근무 인원 부족일
- **연차 거의 소진**: 잔여 1일 이하 직원
